# Хранители Числовых Врат

Обучающая игра-тренажёр таблицы умножения. Один самодостаточный файл, работает офлайн, прогресс — в `localStorage` (у каждого ребёнка свой профиль).

## Структура

Исходник разбит на модули в `src/`. Игра собирается в единый `index.html` (артефакт и запуск через `file://` требуют одного файла).

```
src/
  shell.html        каркас страницы с плейсхолдерами /*__STYLES__*/ и /*__SCRIPT__*/
  styles.css        все стили
  00-data.js        мир, профили, константы, defaultState
  10-state.js       игроки (профили детей), save/load, реестр
  20-core.js        утилиты, звук, числа словами
  30-srs.js         факты, интервальное повторение, подсказки
  40-engine.js      движок экрана вопроса (askFact, таймер, «один выстрел»)
  50-onboarding.js  выбор игрока, титул, профили, вступление, диагностика, плейсмент
  60-map.js         карта мира
  70-practice.js    очередь повторений, Марафон, Испытание Мастера
  80-land.js        земля и врата
  85-boss.js        боссы, Бой-молния
  88-challenges.js  портал, Уголок Забвения, «Без магии», финальный Хаос
  90-hub.js         Кузница, Мастерская, Зал Славы, Панель Магистра
  99-boot.js        запуск + регистрация service worker
  smoke.js          смоук-тесты (в сборку игры НЕ входят)
```

Модули — это обычные скрипты, делящие общую область видимости; порядок сборки = порядок в игре. `build.mjs` просто конкатенирует их, поэтому поведение идентично.

## Сборка

```
node build.mjs
```

Собирает:
- `index.html` — играбельный единый файл;
- `dist/game.html` — урезанная версия для публикации артефактом (без внешней обёртки и PWA);
- `dist/bundle.js` — конкатенация JS для тестов (тот же строгий режим, что и в проде).

> Источник истины — `src/`. `index.html` генерируется; **не редактируйте его напрямую** — правьте модули и пересобирайте.

## Тесты

```
node build.mjs
# затем открыть tests.html через локальный сервер (не file://):
python -m http.server 8571   →   http://localhost:8571/tests.html
```

Заголовок вкладки покажет `✅ 21/21` или `❌`. Прогоняет ключевые инварианты: близнецы, SRS, семантику «верно с первой попытки», «один выстрел», таймаут≠ошибка, гейтинг карты, боссов без пощады, счётчики режимов, изоляцию профилей детей и миграцию старого сейва.

## Прочее

- `sw.js`, `manifest.webmanifest`, `icon-*.png` — PWA (офлайн, установка на планшет). Работают при хостинге по http/https.
- `tools/split.mjs` — одноразовый разбор монолита на `src/` (для истории; повторно не нужен).
- `pravila.html` — печатные правила игры.
